What's inside

Everything a builder needs.
Nothing they don't.

🧾
Dodo Checkout Embed
One-line embed for any web page. Supports INR, USD, EUR. UPI, cards, Klarna, PhonePe. Handles tax and compliance automatically.
🎛️
Usage Credit Engine
Define credit costs per agent action. Subscriptions allocate monthly credits. Pay-as-you-go credit packs. Real-time deduction on every run.
🪝
Webhook Middleware
DodoArc receives Dodo payment events and automatically activates or pauses user access. Retry logic included. Zero manual ops.
◎
x402 USDC Settlement
When your agent calls a paid MCP tool, x402 payment headers settle USDC on Solana automatically. 400ms. No wallet UX for the user.
📊
Unified Revenue Dashboard
See fiat revenue, credit consumption, agent run counts, and Solana settlement receipts — all in one screen. ARR visible from day one.
🔗
Solana Wallet Link
Link a Phantom wallet to receive consolidated USDC settlement. One-click withdrawal. On-chain receipt for every settlement event.
